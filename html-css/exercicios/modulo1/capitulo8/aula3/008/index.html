Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <B> (forma não semântica, não há significado, apenas forma)

Nessa frase temos um termo em negrito usando a tag <STRONG> (forma semântica, há significado)

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <I> (não semântica)

Nessa frase temos um termo em itálico (ênfase) usando a tag <EM> (forma semântica)

Texto Marcado

Nessa frase temos um texto marcado (há o uso de css para alterar a cor da marcação, já que apenas css altera a forma) usando a tag <MARK>

Texto Grande e Pequeno

Estamos criando um texto grande com a tag <BIG> (tag obsoleta) e um texto pequeno com a tag <SMAL> nesse parágrafo

Texto Deletado

Podemos marcar um texto como excluído com a tag <DEL> (deletado) para indicar que deve ser lido, mas não considerado

Texto Inserido

Existe também o texto como inserido (sublinhado) com a tag <U> (forma não semântica)

Podemos marcar um texto comoinserido (sublinhado) com a tag <INS> para dar uma ênfase e indicar que ele foi adicionado depois (forma semântica)

Texto Sobrescrito

Para inserir coisas do tipo X20+3 com a tag <SUP>

Texto Subescrito

Para inserir coisas do tipo H20 com a tag <SUB>
