Testando carga e imagens

Abaixo voce vai ver uma imagem que está na mesma pasta.

Com o código <imc src = "(nome do arquivo)" alt = "(nome alternativo)">

Como por exemplo: <imc src = "logohtml.png" alt = "Logo HTML5">

Podemos também colocar imagens que estão em sub-pastas

Com o código <imc src = "(nome da sub-pasta)/(nome do arquivo)" alt = "(nome alternativo)">

Como por exemplo: <imc src = "imagens/logocss.png" alt = "Logo CSS3">

Também podemos carregar imagens externas

Com o código <imc src = "(link do arquivo)" alt = "(nome alternativo)">

Como por exemplo: <img src = "https://logospng.org/download/javascript/logo-javascript-256.png" alt = "Logo JavaScrip">
