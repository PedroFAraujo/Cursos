UL

O que são

Listas onde a ordem dos itens não importa

Como criar

Use a tag <ul> para delimitar a lista e <li> para identificar cada item.
A tag <ul> tem parâmetro type, a qual configurarmos o tipo de marcados. As opções são:

  • Disc (padrão
  • Circle
  • square
